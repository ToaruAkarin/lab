# Getting Started

# Adding Caffeine tile

Caffeine is a tile-based app. You can only activate or deactivate it by clicking Caffeine tile. Following steps show you how to add Caffeine tile.

  • Pull down status bar
  • Click the pencil icon () or other similar button
  • Add Caffeine tile to quick settings

# Resolving compatibility issues

After adding Caffeine tile to quick settings, try activating it. If it works, you can close this guide now. If not, please refer to Issues.
