# General

This page shows you common problems and their solutions.

Make sure you have followed instructions in User Guide before reading this page.

# Caffeine not working

# Tile can't be lighted

Under most circumstances, this is because the compatibility mode is enabled.

If the compatibility mode is enabled, simply grant permission to modify system settings manually or disable compatibility mode.

Otherwise, try turning off battery optimization for Caffeine and granting Caffeine auto start permission.

If the problem still exists, refer to Exclude from recents.

# Screen still turns off

Please try the following steps:

  • Enable "Compatibility Mode"
  • Grant "Modify system settings" permission
  • Click "Ignore Battery Optimization" option to configure
  • Activate Caffeine again

# The battery optimization page is shown after activating Caffeine

  • Click "Ignore Battery Optimization" option to configure

# Device not sleeping with Caffeine deactivated

Problems like this happens only under compatibility mode. And this is becauce you force stopped Caffeine or removed it from recent apps before disabling it.

This can be easily solved by resetting your screen timeout in system settings.

In addition, Caffeine will try to automatically reset your preferred screen timeout each time you pull down the quick settings panel and see the Caffeine tile. The best results can be achieved by placing the Caffeine tile within the visible range of a single pull-down.

TIP

If you don't want to see Caffeine in recent apps, please enable Exclude from recents.

# Device compatibility

Due to a variety of reasons, Caffeine cannot work well on some ROMs. If solutions on this page didn't solve your problem, please refer to Device Compatibility.

# Caffeine launched after being deactivated from notification

Remove Caffeine from recent apps before activating it.

# Report issues

If all solutions listed above didn't solve your problems, please refer to Feedback.
