# 常见问题

本页面列出了一些常见问题及其解决方案。

在阅读本页之前，请确保你已仔细阅读「使用说明」。

# 咖啡因无法正常工作

# 图块无法被点亮

多数情况下，发生此种问题是由于你启用了「兼容模式」。

如果你启用了「兼容模式」，请手动授予咖啡因「修改系统设置」权限，或是关闭「兼容模式」后再次尝试启动咖啡因。

否则，请尝试为咖啡因关闭「后台限制」和「电池优化」，并授予咖啡因「自启动」权限。

如果问题仍然存在，请参阅「从最近应用中排除」下方文字。

# 屏幕仍会熄灭

请尝试以下步骤：

  • 启用「兼容模式」
  • 授予「修改系统设置」权限
  • 点击「忽略电池优化」进行 配置
  • 再次激活咖啡因

# 在关闭后屏幕仍保持常亮

此种问题只会在「兼容模式」启用时出现。在咖啡因运行时，如果你强制关闭了咖啡因，此问题便会出现。

重新设置系统中的「屏幕超时」就能解决此问题。

此外，咖啡因会尝试在你每次下拉快速设置面板，看到咖啡因图块的时候自动重设你的偏好屏幕超时。将咖啡因图块置于单次下拉的可见范围内可以达到最佳效果。

提示

如果你不想在最近应用中看到咖啡因，请启用「从最近应用中排除」。

# 设备兼容性

由于各种各样的原因，咖啡因不能在某些定制系统中完全正常工作。如果本页面所提供的方法仍不能使咖啡因正常工作，请参阅「设备兼容性」。

# 点按通知中的关闭按钮后跳转至咖啡因设置

请在激活咖啡因之前将其从最近应用中移除。

# 反馈问题

如果上述所有方法都不能解决你的问题，请参阅「反馈」。
