# 与其他应用联动 新增

在 1.4.0 版本中，咖啡因新增了一个用于切换状态且可供外部调用的 Activity。启动此 Activity 将切换咖啡因的状态。

这意味着，咖啡因可以和外部应用配合，以实现自动化（例如：打开某个应用的同时，自动启用咖啡因）或是其他效果。

要启动此活动，请用以下信息创建一个 Intent：

  • 包名：moe.zhs.caffeine
  • 活动：moe.zhs.caffeine.ToggleActivity
  • 附加 Action：android.intent.action.VIEW
