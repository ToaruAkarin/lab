# 配置

此处列出的是咖啡因中所有配置项及其说明。

这些配置项的默认值均为空或禁用。

# 常规

# 定时关闭

提示

此项仅在通过快捷设置图块启用咖啡因时生效。

启用咖啡因后保持屏幕唤醒的时间。

设置为「动态」后，可以通过反复点击咖啡因图块来更改定时关闭时间。

# 自定义图块标题

显示于咖啡因图块下的标题。

# 自动关闭

手动关闭屏幕时自动关闭咖啡因。

# 允许变暗

提示

此项不能与「兼容模式」同时启用。

在保持屏幕唤醒的前提下，允许屏幕亮度降低。

# 高级

# 忽略电池优化

提示

在 Android 14+ 下，必须配置此选项。

部分系统的省电策略会导致咖啡因出现问题。

在点击此选项后，选择「所有应用」，然后找到咖啡因，并对其关闭电池优化。

如果你无法从列表中找到咖啡因，请先关闭「隐藏图标」开关。

在正确配置后，此选项会被隐藏。

# 兼容模式

提示

此项仅在咖啡因取得「修改系统设置」权限后有效。

通过修改系统设置的方式来实现保持屏幕唤醒。

这对默认情况下不能使用咖啡因的设备有帮助。

# 仅使用标题

提示

此项仅适用于 Android 10+，且在 MIUI 中无法被关闭。

仅使用图块的标签来显示定时关闭时间。

Android 10 为快速设置中的图块引入了副标题。咖啡因针对这个特性进行了适配，默认会同时使用双行图块标题。但是在部分定制系统（例如 MIUI）上，这项特性被去除了。这导致定时关闭所剩的时间无法显示。此时启用此项，可以将时间显示到图块的标题上。

# 隐藏图标

提示

在没有安装为系统应用时，此项不适用于除 MIUI 之外的 Android 10+。

在启动器中隐藏咖啡因的图标。

启用此项后请通过长按咖啡因图块或点击咖啡因发出的通知来打开咖啡因。

从 Android 10 开始，用户应用无法隐藏启动器图标。不过在某些定制系统（例如 MIUI）上，它仍然有效。所以在咖啡因没有被安装为系统应用时，此项在 Android 10+ 中仅为 MIUI 开放。

如果有条件，你也可以将咖啡因安装为系统应用来解锁这个选项。

# 从最近应用中排除

提示

切换此选项后需要重启应用。

在最近应用中隐藏咖啡因。

部分定制系统有着严格的后台限制，在最近任务中关闭咖啡因后会导致图块响应延迟或无法响应用户的操作。此时，请先将咖啡因于最近应用列表中锁定，然后启用这个选项并重新启动咖啡因。
