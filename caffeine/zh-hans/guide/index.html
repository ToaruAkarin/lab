# 添加开关图块

咖啡因的唯一开关就是快速设置中的图块。以下步骤将为你展示如何添加咖啡因图块。

  • 下拉状态栏
  • 点击铅笔图标（）或者其它类似按钮
  • 将咖啡因图块加入快速设置

# 解决兼容性问题

在将咖啡因加入快速设置后，请尝试启用它。如果它正常工作，你就可以关掉这篇文章了。否则，请参阅「问题」。
