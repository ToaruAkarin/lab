# 开始

感谢安装咖啡因，本页将告诉你如何使用它。

警告

长时间保持屏幕唤醒可能会损伤屏幕。

# 使用

咖啡因目前提供了数种使用方法，你可以根据实际情况选择最合适的方法来使用它。

# 使用快捷设置图块

咖啡因的主要开关是快捷设置中的图块。以下步骤将为你展示如何添加咖啡因图块：

  • 下拉状态栏
  • 点击「」或其他类似按钮
  • 将咖啡因图块加入快速设置

# 使用快捷方式 1.4.0+

在 Android 7.1 及以上版本中，除了快捷设置图块，你还可以通过快捷方式使用咖啡因。

通常来说，在启动器中长按咖啡因的应用图标，再点击「切换」选项即可切换咖啡因的状态。

在大部分系统中，你还可以通过按住选项右边的「」来将其添加到主屏幕。

# 使用外部方式 1.4.0+

请参考「与其他应用联动」。

# 解决兼容性问题

在将咖啡因加入快速设置后，请尝试启用它。如果它正常工作，你就可以关掉这篇文章了。否则，请参阅「问题」。
