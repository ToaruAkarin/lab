# 小米（MIUI）

由于严格的后台限制，在 MIUI 中可能存在图块无法被点亮的问题。你需要手动调整省电策略：

  • 暂时关闭应用的「隐藏图标」功能
  • 打开「手机管家」
  • 进入「省电与电池」
  • 点击右上角的「」图标
  • 选择「应用智能省电」
  • 点击「咖啡因」
  • 选择「无限制」
