# 设备兼容性

咖啡因针对部分系统进行了一些预配置，以此确保其能够在你的设备上正确工作。

# 设备（ROM）

# 小米（MIUI）

  • 强制启用「兼容模式」
  • 需要点击「忽略电池优化」进行配置
  • 取消「隐藏图标」功能对 Android 版本的限制

# 三星（One UI）

# Android 12+

  • 强制启用「兼容模式」
  • 需要点击「忽略电池优化」进行配置

# OPPO / OnePlus（仅 Android 12+）/ realme（OPPO ROM）

# 较新系统版本

由于严格的后台限制，较高版本的 OPPO ROM 会导致咖啡因无法工作。

来自用户 Rory Church 的解决方案：

  • 禁用「定时关闭」、「允许变暗」和「兼容模式」
  • 需要点击「忽略电池优化」进行配置

# 较老系统版本

对于较老版本的 OPPO ROM，咖啡因仅能保持屏幕在较暗亮度下唤醒。

  • 强制启用「允许变暗」
  • 不可启用「兼容模式」

# 专家模式

如果你不希望使用预配置选项，你可以启用「专家模式」来进行手动配置。

要启用它，请点击「版本号」七次，然后点击「专家模式」。
