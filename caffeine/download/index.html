# Download

Currently, Caffeine is compatible with devices running Android 7.0+.

You can choose between release build and canary build according to details listed below. If you wish to switch between these versions, you should uninstall your current version first.

Also, you should avoid downloading packages from sources not mentioned in this page.

# Release Build

Manually built stable versions, usually released after several code changes.

Less frequent updates with higher stability. Suitable for most users.

You can get release builds from Google Play (opens new window).

# Canary Build

Automatically built versions, released after each code change.

Lower stability with frequent updates. Suitable for users who want early access to leading-edge features.

You can get canary builds from App Center (opens new window).
